Daftar perintah GiT
1. Pada bagian main branch, pergi ke bagian branch.
2. Buat cabang baru dengan menekan tombol new branch.
3. Masukan nama cabang baru, lalu tekan create new bracnh.
4. Klik pada cabang yang baru dibuat untuk pergi ke bagian cabang tersebut.
5. Klik add file untuk menambahkan file lalu beri nama file sesuai dengan urutan tugas yang diberikan.
6. Edit file dengan memsukan teks bebas lalu tekan commit changes untuk menyimpan perubahan.
7. Tekan tombol "Compare and Pull Request" untuk melakukan merge ke cabang utama.
8. Tekan Create Pull request lalu konfirmasi.
